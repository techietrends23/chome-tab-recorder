About Save Locations

Chrome extensions cannot choose arbitrary folders on your filesystem. Files are downloaded to your browser's default Downloads directory. You can optionally specify a subfolder name to organize recordings.

If you enable "Ask where to save", Chrome will show a Save As dialog for each recording.
