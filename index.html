Quick start

Install in a couple of minutes

1

Pick your install path

Try the CRX first. If Chrome rejects it, use the ZIP package instead and unpack it on your machine.

2

Open Chrome extensions

Go to chrome://extensions and turn on Developer Mode in the top right corner.

3

Install the extension

For ZIP: unzip it, click Load unpacked, and select the extracted folder. For CRX: drag it into the extensions page if your Chrome build allows it.

4

Start recording

Open the extension, choose export format and quality, click Start, then use the in-page toolbar to annotate while you record.

How it works

A clean recording flow for real work

1. Launch the recorder

Click the extension icon, confirm format and quality, then start recording the active Chrome tab.

2. Annotate on the page

Use the floating toolbar for pen, highlighter, text, eraser, click spotlighting, and theme switching.

3. Pause or stop anytime

Use the popup or the page toolbar controls. The recording engine runs offscreen so the popup can be closed safely.

4. Download instantly

Your finished recording is saved to Chrome Downloads, with optional subfolder organization from settings.

Need-to-know

Install notes for your users

Why provide both CRX and ZIP?

CRX is convenient for compatible Chromium environments. ZIP is the reliable fallback for Chrome Developer Mode installs.

Can users install this like a normal store extension?

Not yet. For the smoothest one-click install on Chrome, publish the extension to the Chrome Web Store later. This page gives users the fastest sideload path right now.

Where do recordings go?

Files are downloaded through Chrome into the browser Downloads folder, optionally inside a subfolder you configure in settings.
